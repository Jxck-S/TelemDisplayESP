# TelemDisplayESP
TelemDisplayESP is a project  for displaying flight data on 7 segment displays based on the ESP8266. 


## Parts
- ESP8266
- 4, 7 segment displays, using TM1637 controllers
- Jumpers, Power supply
- Board to hold it all together
